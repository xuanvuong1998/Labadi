Author: me and @minhtan1198
Idea: mainly by Kahoot and partially by Confetti
Platform: .NET
Framework: ASP.NET
Model: MVC (Model, View, Controller)
Database: SQL Server 2014
Database connection technology: ADO.NET
Library: jquery, fontawesome, bootstrap, signalR...
Techniques: lazyloading
IDE: Visual Studio 2017
Accessible web site link: https://labadi.azurewebsites.net

Description: The web application provides CRUD operations for managing quizzes and allows joining a quiz by multiple
players. After starting a quiz, the host side can send questions to players side and vice versa, the players
send selected answer to the host; Then the host analyzes and give final results to all joinned players.
